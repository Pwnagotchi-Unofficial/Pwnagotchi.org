download plugins easily|

Are you looking for community made Pwnagotchi plugins? We got a long list of the most used plugins.

Just place the plugins you want to use into your custom plugin directory.
Do not forget to enable your custom plugins in your config.toml!


GPS_more

Modified version of stock gps.py. After loading, it will update the GPS on set epochs until it gets a fix, then it just updates for handshakes. That way you can confirm it´s working before you get a handshake. Optionally saves all gps locations to a file (epochs and handshakes), like a breadcrumb trail.

GPS-plus

Displays gps coordinates, like the original plugin, but with added options. Made to go with memtemp-plus.

Net-Pos

Saves a json file with the access points nearby whenever a handshake is captured. When internet is available the files are converted in geo locations using the Mozilla LocationService API. Useful if you havnt got a GPS-module.

f0xtr0t

A plugin that shows a openstreetmap with positions of ap-handshakes in your webbrowser. Based on the original webgpsmap plugin.

memtemp-plus

A plugin that will display memory/cpu usage and temperature. Based on original memtemp plugin with added features.

Dashboard2

Dashboard2 plugin is a consolidation of the clock, deauth counter, memtemp, pivoyager and added few features such as cracked handshake counter and internet status.

Pwnmenu

This plugin is a simple popup menu to run scripts from.

Display Version

A plugin that will display the Pwnagotchi version left of the current mode.

Internet Connection

A plugin that displays the Internet connection status on the pwnagotchi display.

Experience

Get exp every time a handshake gets captured, displays exp and lvl on ui.

Age

A plugin that will add age and strength stats based on epochs and trained epochs.

AircrackOnly

confirms pcap contains handshake/PMKID or deletes it

handshakes-dl-hashie

Download captured handshakes from the web-ui, but the handshakes automatically get converted to hashcat format with hashie-hcxpcapngtool.py.

Note: you need hashie-hcxpcapngtool.py & handshakes-dl-hashie.py in your plugin folder for it to work.

DiscoHash

DiscoHash extracts hashes from pcaps (hashcat mode 22000) using hcxpcapngtool, analyses the hash using hcxhashtool and posts the output to Discord along with any obtained GPS coordinates.

better_quickdic

Runs a quick dictionary scan against captured handshakes, nut unlike the original you can upload wordlist files. Cracked hanshakes will be stored as [essid].pcap.cracked.

Crackhouse

A plugin to display closest cracked network & its password.

Display Password

A plugin to display recently cracked passwords from the upload plugins.

My Cracked PW

A plugin to grab all cracked passwords and creates wifi qrcodes and a wordlist which can be used for the quickdic plugin. It stores them in the home directory.

Educational

A plugin to automatically authenticate to known networks and perform internal network recon.

Banthex

This plugin automatically uploads handshakes to https://banthex.de/wpa/.

OnlineHashCrack

This plugin automatically uploads handshakes to https://onlinehashcrack.com.

WpaSec

This plugin automatically uploads handshakes to https://wpa-sec.stanev.org.

HashesPwnagotchi

This plugin uploads handshakes to https://hashes.pw.

Pisugar 3 Improved

The PiSugar 3 Plugin displays the battery percentage as well as the charging status on your Pwnagotchi's UI.

Pisugar2

A plugin that will add a voltage indicator for the PiSugar 2.

External WiFi

Enable/disable the external wifi with ease.

Fix Region

Lets you change the iw region to unlock channels.

Hulk

This plugin activates Hulk mode and Hulk is always angry.
